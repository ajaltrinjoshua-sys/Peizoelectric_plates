⚡ Piezoelectric Energy Harvesting System

By Altrin Joshua

🔹 Introduction

This project demonstrates how mechanical energy from footsteps can be converted into electrical energy using piezoelectric plates.

⚙️ Working Principle

When pressure is applied to piezoelectric plates, electrical voltage is generated. This is called the Piezoelectric Effect.

🔌 Components Used

  • Piezoelectric Sensors
  • LED / Display
  • Wires
  • Capacitor / Battery

🚀 Unique Feature

This model includes a smart step-based energy generation system with real-time output demonstration.

🌍 Applications

  • Smart Roads
  • Railway Stations
  • Shopping Malls
  • Renewable Energy Systems
